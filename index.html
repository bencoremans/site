🔐 SecretBox

Encrypteer en decrypteer tekst veilig in je browser - niets wordt verzonden!

Hoe werkt het? Voer je geheime tekst en wachtwoord in. De tekst wordt versleuteld met AES-256-GCM en blijft in je browser - niets wordt naar een server gestuurd!
✅ Tekst succesvol versleuteld! Kopieer de encrypted tekst hieronder.
Hoe werkt het? Plak de encrypted tekst en voer het wachtwoord in om de originele boodschap te zien.
✅ Tekst succesvol gedecrypteerd!

100% Privacy: Alle encryptie gebeurt lokaal in je browser met Web Crypto API (AES-256-GCM).
